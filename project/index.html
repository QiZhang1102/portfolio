Music Recommendation System （Sep 2024 - Jan 2025）

Objective: Design a personalized recommendation system to suggest Taylor Swift songs based on audio emotion and listener preferences.

Actions: Cleaned and analyzed audio feature data (e.g., energy, valence, danceability) using Python (pandas, NumPy); engineered novel features such as "potency" to quantify emotional intensity; implemented a similarity-based recommendation algorithm using Euclidean distance over selected audio dimensions.

Results: Achieved over 90% perceived relevance in manual evaluation; significantly improved interpretability and alignment with listener mood compared to baseline approach.

Old Friends Sitcom Reboot Analysis （Sep 2024 - Jan 2025）

Objective: Design a strategy for rebooting the sitcom Friends by analyzing patterns in character dynamics, director impact, and episode performance.

Actions: Extracted and cleaned multi-table episode-level data (writers, directors, dialogues, emotions); implemented statistical analysis (bootstrapping, confidence intervals, hypothesis testing) to evaluate director performance and gender line balance.

Results: Identified Kevin S. Bright as the statistically superior director with 99% confidence; confirmed no significant gender imbalance (p = 0.0737), guiding fair character scripting in the reboot plan.

Power Outage Impact Analysis （Mar 2025 – Jun 2025）

Objective: Analyzed how climate and electricity prices influence power outage duration.

Actions: Cleaned and structured 1,400+ outage records; conducted statistical analysis and feature engineering; built and tuned a Random Forest model to predict outage duration; created interactive visualizations to explore regional and temporal trends.

Results: Reduced model RMSE by 7% through improved feature design; uncovered significant regional disparities in outage length.
